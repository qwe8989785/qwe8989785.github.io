Hi,我是Wayne!

我來自台南，自幼便深陷電腦的世界，這深遠地影響了我未來的求學和職涯方向，使我一直與資訊科技領域保持著密不可分的聯繫。
在高中時，我進一步探索了網路管理這個領域。在老師的悉心指導下，我成功地獲得了教育部及勞動部的技能競賽獎項，也借此優勢成功推甄進入了高雄科技大學。
在大學階段，我持續充實自己的程式能力，將Python視為我的主要武器，用以進行專題開發和其他專案的撰寫。
除了程式技能外，我積極參與了籃球系隊。在這個過程中，我學到了團隊合作、溝通和互助的重要性，並建立了一群可靠的朋友。
經過大學階段的種種磨練，我目前擅長後端開發，使用Python的FastAPI和Django框架撰寫API，同時也涉獵機器學習和影像辨識等技術。然而，我深信學無止境，因此在閒暇時繼續探索新技術，例如Ruby on Rails、React.js等。

主要開發工具為Python FastAPI、Django Framework，PostgreSQL。

作品

1. Fiesta 學生活動票卷販賣系統後端API:

Fiesta是基於學生族群的活動售票系統，其中後端功能包含會員系統、活動、票卷等等，方便的功能包含產生入場QRCode驗票，發送會員驗證信，串接金流。整個後端架構是用Flask Framework + mySQL 實作而成。資料庫部分也是本人親自設計，遵守正規原則。

Github連結

2. 高科大英文聽說讀寫專案系統後端API:

本系統為一英文練習平台，在上面可以練習聽說讀寫，後端API功能實作Spacy之相關功能，使用Django作為主體架構，搭配mySQL + nginx + uwsgi 以完成完整服務。

Github連結

3. 狗狗辨識系統:

本系統為了練習影像辨識而建置，在上面可以上傳你的狗狗照片，我們將告訴你你上傳的照片中的狗狗是甚麼狗狗。採用Django建置，搭配yolo及ResNet50模型進行訓練，以達成辨識效果。

參考資料:
Stanford Dogs Dataset Stanford Dogs Resnet50
Github連結

4. FastAPI 簡易庫存系統:

本系統是撰寫IT邦鐵人賽時所一起建構的簡易範例，為了可以讓看教學的人可以更清楚的學習，所以使用庫存系統做了一個簡單的範例。

鐵人賽連結:
FastAPI 入門30天
Github連結

5. Saltar 社交玩樂地圖App:

我們致力於讓人們產生更多線下社交的機會及場景，我們利用遊戲化的構想，結合活動社群的特性，將索然無味的現實世界打造成類開放世界RPG的線下活動社群，並利用許多卡牌收集、交換、養成要素，使人們有更強烈的動機參與線下社交，並且沈浸在線下社交的樂趣當中。

本人負責後端API開發，使用FastAPI框架，並串接PostgreSQL資料庫，同時也使用 aws 的 dynamo db 及容器服務。
